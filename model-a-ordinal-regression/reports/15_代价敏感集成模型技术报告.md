## 一、项目背景与目标

### 1.1 背景

压力性损伤（Pressure Injury, PI）是肿瘤患者住院期间常见的并发症之一，严重影响患者生活质量和治疗效果。基于 Braden 量表的风险评估是临床预防的重要手段，但传统人工评估存在主观性强、效率低等问题。本项目旨在开发基于患者临床数据的机器学习模型，实现压力性损伤风险的自动化预测。

### 1.2 前期工作回顾

在 v1.0 技术报告（2026-05-07）中，团队完成了基线模型构建和 SMOTE 过采样优化实验，主要结论为：

- 基线有序逻辑回归 Kappa=0.078，几乎无法识别风险患者
- SMOTE 优化后最优 Kappa=0.210，从"极弱"提升至"弱"一致性
- 但 Kappa 仍处于较低水平，各类别识别不均衡

### 1.3 本期目标

在前期 SMOTE 优化基础上，引入代价敏感学习和模型集成策略，进一步优化分类性能。同时针对不同临床场景提供差异化模型方案。

## 三、建模方法

### 3.1 整体技术路线

本期在 v1.0 的 SMOTE 过采样基础上，引入以下 5 项核心创新：

```
原始数据 → 编码 → 分层划分(70:30) → 标准化 → [训练/验证子集划分]
                                              │         │
                                         SMOTE重采样  阈值优化
                                              │         │
                                         代价敏感训练  验证集评估
                                              │
                                         软投票集成
                                              │
                                         测试集评估
```

### 3.2 核心创新一：序数距离代价矩阵

传统的类别不平衡处理（如逆频率加权）仅关注样本数量的不均衡，忽略了有序分类问题中"错分距离"的语义差异。例如：

- 将高风险（3）误判为无风险（0），远比误判为中风险（2）严重
- 将低风险（1）误判为中风险（2），影响较小

据此设计了基于等级间距离的序数代价矩阵：

```
cost(i, j) = |i - j|^2
```

|  | 真实0 | 真实1 | 真实2 | 真实3 |
|--|-------|-------|-------|-------|
| **预测0** | 0 | 1 | 4 | **9** |
| **预测1** | 1 | 0 | 1 | 4 |
| **预测2** | 4 | 1 | 0 | 1 |
| **预测3** | **9** | 4 | 1 | 0 |

代价矩阵决定了样本权重：对某个类别，其被误判的平均代价越高，该类别样本在训练中获得越高的权重。

### 3.3 核心创新二：SMOTE 变体优选

在前期 SMOTE 的基础上，进一步引入两种变体进行对比：

| 变体 | 策略 | 特点 |
|------|------|------|
| **SMOTE** | 在少数类 K 近邻间插值生成样本 | 基础过采样 |
| **Borderline-SMOTE** | 仅在边界区域少数类样本附近生成 | 聚焦决策边界 |
| **SMOTE-ENN** | SMOTE 后用 ENN 清除噪声样本 | 更干净的训练集 |

过采样策略统一采用 moderate 策略：将每个少数类增采样至多数类的 40%。

### 3.4 核心创新三：代价敏感训练

基于代价矩阵为每个训练样本计算权重，在模型训练时引入代价信息。设计了三种权重策略：

**（1）逆频率权重（inverse frequency）**

传统的类别不平衡加权方式：

```
w_class = N_total / (K × count_class)
```

**（2）序数代价权重（ordinal cost）**

基于代价矩阵的行均值计算每个类别的权重：

```
w_class = avg_{j != class} cost(class, j)
```

对于4类问题：无风险=4.67，低风险=2.00，中风险=2.00，高风险=4.67。高风险和无风险获得更高权重。

**（3）混合权重（combined）**

将两种策略以 alpha 混合：

```
w = alpha × 逆频率 + (1 - alpha) × 序数代价    (alpha=0.5)
```

混合权重既考虑了类别不平衡（逆频率），又考虑了错分严重程度（序数代价）。

### 3.5 核心创新四：概率偏移阈值优化

默认的 argmax 决策规则将概率最高的类别作为预测结果，等价于隐含了"各类别阈值相等"的假设。在类别不平衡问题中，这往往导致模型过度偏向多数类。

本方案在验证集上搜索一个全局概率偏移量 offset：

```
adjusted_prob[i] = prob[i] + offset × (K-1-2i) / (K-1)
```

- offset > 0：模型更倾向于预测低风险类别（保守策略）
- offset < 0：模型更倾向于预测高风险类别（敏感策略）

搜索范围 [-0.3, 0.3]，采用粗搜索 + 精搜索两轮迭代。

### 3.6 核心创新五：软投票集成

将多个模型的概率预测取平均，再应用阈值偏移进行决策：

```
ensemble_prob = mean(model_1_prob, model_2_prob, ..., model_n_prob)
```

设计了三种集成策略：

| 集成策略 | 基模型 | 特点 |
|----------|--------|------|
| 全部软投票 | 全部9个单模型 | 最大化多样性 |
| **Top3 + 阈值** | Macro F1 最高的3个基模型 | 精选最优模型 |
| XGB + LGB 混合 | 全部XGBoost和LightGBM模型 | 利用树模型互补性 |

### 3.7 数据划分流程

```
全部数据 (100,504)
    │
    ├── 分层划分 (70:30, random_state=42)
    │       │
    │       ├── 训练集 (70,352)
    │       │       │
    │       │       └── 分层划分 (85:15, random_state=123)
    │       │               │
    │       │               ├── 训练子集 (59,799) → SMOTE → 训练模型
    │       │               └── 验证子集 (10,553) → 阈值优化
    │       │
    │       └── 测试集 (30,152) → 最终评估（不参与任何训练决策）
    │
    └── 输出: 混淆矩阵 + ROC曲线 + 全量指标 + 对比图表
```

验证集独立于训练和测试，仅用于阈值优化，避免信息泄露。

## 四、评估指标体系

沿用 v1.0 的多维度评估指标体系：

### 4.1 核心指标

| 指标 | 说明 | 评判标准 |
|------|------|----------|
| **Cohen's Kappa** | 衡量预测一致性，校正随机一致性 | <0.20 极弱, 0.21-0.40 弱, 0.41-0.60 中等 |
| **多分类 AUC (macro)** | One-vs-Rest 策略，各类别 AUC 的算术平均 | 综合衡量概率排序能力 |

### 4.2 辅助指标

| 指标 | 说明 |
|------|------|
| **Macro F1** | 各类别 F1 的算术平均，平等对待每个类别 |
| **Weighted F1** | 按样本量加权的 F1 平均 |
| **G-mean** | 各类别召回率的几何平均，衡量均衡识别能力 |
| **各类别 Precision / Recall / F1** | 逐类别详细指标 |

## 八、结果分析与讨论

### 8.1 序数代价权重的作用

纯序数代价权重（模型3、4）呈现出鲜明的"两极化"特征：

- **优势端**：高风险Recall显著提升（49.7%/39.0% vs 其他模型的15%-19%）
- **劣势端**：低风险Recall骤降（6.7%/7.1%），Macro F1下降

原因在于代价矩阵的结构：高风险和无风险的序数代价权重均为 4.67，而低风险和中风险仅为 2.00。模型在训练中更关注两极类别，导致中间类别被忽视。

### 8.2 混合权重的优势

混合权重（alpha=0.5）在各维度上表现更为均衡：

- 高风险Recall虽不及纯序数代价，但低风险和中风险的召回率大幅改善
- Kappa 和 Macro F1 均优于纯序数代价
- 在 LightGBM + Borderline-SMOTE 的组合下，Macro F1 达到全部实验最高值（0.3873）

结论：混合权重是更实用的策略，在总体一致性和风险识别能力之间取得了更好的平衡。

### 8.3 SMOTE 变体的差异化表现

| SMOTE变体 | 最优搭配 | 特点 |
|-----------|---------|------|
| SMOTE | XGBoost + 序数代价 | 配合序数代价权重达到最高AUC |
| Borderline-SMOTE | LightGBM + 混合 | 达到最高Macro F1和Kappa |
| SMOTE-ENN | XGBoost + 混合 | 最高低风险和中风险Recall |

Borderline-SMOTE 聚焦决策边界的策略与 LightGBM 的叶子分裂机制协同效果最好，而 SMOTE-ENN 的噪声清洗效果有利于 XGBoost 对少数类的识别。

### 8.4 阈值优化的关键作用

阈值优化在本实验中发挥了至关重要的作用。以高风险预警模型为例：

| 方式 | 高风险Recall | Kappa |
|------|-------------|-------|
| 默认 argmax | 19.8% | 0.185 |
| 阈值优化 | 49.7% | 0.248 |

概率偏移阈值优化通过微调类别边界，使模型在不改变训练过程的情况下显著改善了对高风险样本的识别。这一步在集成的 Top3 模型中同样有效，是 Kappa 达到 0.279 的关键因素之一。

### 8.5 集成学习的增益

Top3 集成（Macro F1 最高的 3 个基模型 + 阈值优化）在 Kappa 上达到最优（0.2788），对比各基模型：

| 模型 | Kappa | Macro F1 |
|------|-------|----------|
| 基模型1: LGB+BorderSMOTE+混合 | 0.2751 | 0.3873 |
| 基模型2: LR+SMOTE+混合权重 | 0.2635 | 0.3860 |
| 基模型3: LGB+SMOTE+混合权重 | 0.2649 | 0.3808 |
| **集成: Top3+阈值** | **0.2788** | 0.3859 |

集成在 Kappa 上超越了所有基模型，但 Macro F1 略低于最佳基模型。这表明集成主要通过减少极端错误分类来提升整体一致性，而非追求单一指标的最优化。

### 8.6 数据层面的局限

当前最优 Kappa=0.279 仍处于"弱"一致性水平，根本原因在于数据结构性限制：

1. **类别极端不平衡**：中风险和高风险样本合计仅 1.1%，即使序数代价权重和阈值优化能提升召回率，Precision 仍受限于噪声比例
2. **特征区分力不足**：24 个临床特征在区分"低风险 vs 中风险 vs 高风险"的细粒度差异时，信息量有限
3. **交叉类别重叠**：不同风险等级的患者在临床特征上的分布存在较大重叠，增加了分类难度

## 九、结论与后续建议

### 9.1 主要结论

1. **序数距离代价矩阵有效**：序数代价权重使高风险Recall从基线的0%提升至49.7%，验证了"距离越远代价越高"的假设在临床分类中的合理性

2. **混合权重优于单一策略**：混合权重（逆频率 + 序数代价）在 Kappa 和 Macro F1 上均优于纯逆频率或纯序数代价，是最实用的权重策略

3. **阈值优化至关重要**：概率偏移阈值优化是提升 Kappa 的关键因素之一，也是高风险召回率从19.8%跃升至49.7%的核心原因

4. **集成学习提供增量提升**：Top3 集成 + 阈值优化达到全部实验最优 Kappa（0.279），较 v1.0 提升 33%

5. **不同场景需选用不同模型**：综合评估推荐集成Top3模型，高风险筛查推荐XGB+序数代价模型，两者服务于不同临床需求
